# minimalist_stock_inventory
Projet à vocation open source pour gérer les petits inventaires de stock (si vous avez 10k références et 200 mouvements par jour c'est pas ça qu'il vous faut)

# Mise en place 

```chmod +x main.sh```

# Lancement

```./main.sh```
